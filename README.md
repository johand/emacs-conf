## Instalacion

**clonar repositorio**

<code>git://github.com/johand/emacs-conf.git</code>


**instalar submodulos**

<code>cd emacs-conf</code>

<code>git submodule init && git submodule update</code>


**copiar fichero y directorio**

<code>cp emacs ~/.emacs</code>

<code>cp -r emacs.d ~/.emacs.d</code>
